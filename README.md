# Noted （not famous）

A lightweight, privacy-focused, and serverless note-taking application built entirely on web technologies. 

**Noted** is designed to be fast, minimal, and secure. It runs completely on the client side with no backend dependency, meaning your data never leaves your device.

---

## 🚀 Live Access

You can use Noted instantly in your web browser without any registration or setup:

* **Primary Link:** [https://notteed.netlify.app/](https://notteed.netlify.app/)

## ✨ Features

- **No Account System:** No registration, no logins, and no tracking. Start writing the moment you open the app.
- **Pure HTML/Client-Side Execution:** Fully compiled and running locally in your browser. Complete data privacy by design.
- **Offline & Desktop Support:** Can be downloaded and run as a standalone application on your local machine.
- **Continuously Updated:** Active development with regular feature enhancements and performance tweaks.

## 📦 Download & Installation

If you prefer using Noted as a desktop application rather than in a browser, you can download the latest standalone version from the **Releases** section of this repository.

*Note: The application will automatically save your data locally, ensuring full functionality even when offline.*

## 🛠️ Built With

- **HTML5** - Structuring the application
- **CSS3** - Styling and layout
- **JavaScript (Vanilla)** - Client-side logic and local data handling
